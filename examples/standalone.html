Google Calendar Widget Demo

Instructions

To use this demo, you need a Google API key with the Google Calendar API enabled.

  1. Create a Google API key in the Google Cloud Console
  2. Enable the Google Calendar API for your project
  3. Enter your API key below
  4. Use a public calendar ID (like a holiday calendar)

Note: For security reasons, you should restrict your API key in the Google Cloud Console.
